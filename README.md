# MedVault - Smart Pharmacy Platform

A premium dark-mode healthcare web app.

## Live Demo

https://eloquent-crostata-1c19ad.netlify.app/login.html

## Features
- Google Sign-In
- Medicine Shop
- Doctor Consultations
- Lab Test Booking
- Prescription Management
- Smart Dose Timer
- Drug Toxicity Predictor
- PWA (installable on mobile)

## Tech Stack
HTML + CSS + Vanilla JavaScript
